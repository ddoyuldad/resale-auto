  📦 매입건 자동화 프로그램 v2.1  —  실행 가이드
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━


┌─────────────────────────────────────────────────┐
│  ▶  실행 방법  (이것만 알면 됩니다!)             │
│                                                 │
│     [웹앱실행.bat] 파일을  더블클릭              │
│                                                 │
│     → 자동으로 패키지 설치 및 서버 시작          │
│     → 브라우저에서 http://localhost:8080 접속    │
└─────────────────────────────────────────────────┘


════════════════════════════════════════════════════
  1. 사전 준비 (처음 한 번만)
════════════════════════════════════════════════════

  ① Python 설치
     → 다운로드: https://www.python.org/downloads
     → 설치 시 ★ "Add Python to PATH" 반드시 체크! ★
     → 설치 완료 후 컴퓨터 재시작 권장

  ② 이 폴더(매입자동화)를 원하는 위치에 복사
     예시: C:\Users\내이름\Desktop\매입자동화


════════════════════════════════════════════════════
  2. 실행 순서
════════════════════════════════════════════════════

  [1/3] 패키지 자동 설치    (최초 실행 시 1~3분 소요)
  [2/3] 설치 상태 확인
  [3/3] 서버 시작 → 브라우저 자동 열림

  ※ 검은 창(명령 프롬프트)은 절대 닫지 마세요!
     닫으면 프로그램이 종료됩니다.


════════════════════════════════════════════════════
  3. 주요 기능
════════════════════════════════════════════════════

  Step 1  이미지 분석 & 폴더 자동 분류
          → Claude AI가 매입 이미지를 읽어 제품별 정리

  Step 2  엑셀 자동 생성
          → 썸네일 + 제품 정보 포함 엑셀 파일 생성

  Step 3  네이버 최저가 자동 조회
          → 네이버 쇼핑 API로 시장 최저가 수집

  Step 4  상세페이지 HTML 자동 생성

  Step 5  쿠팡 등록 데이터 준비


════════════════════════════════════════════════════
  4. 문제 해결
════════════════════════════════════════════════════

  ❌ 'python'은 인식되지 않는 명령
     → Python 재설치 후 "Add Python to PATH" 체크

  ❌ 브라우저가 자동으로 안 열림
     → 직접 주소창에 입력: http://localhost:8080

  ❌ 화면이 변경되지 않음
     → Ctrl + Shift + R  (강제 새로고침)

  ❌ pip 설치 실패
     → 인터넷 연결 확인 후 다시 실행

  ❌ 프로그램이 갑자기 종료됨
     → 웹앱실행.bat 다시 더블클릭


════════════════════════════════════════════════════
  5. API 키 변경 (필요한 경우)
════════════════════════════════════════════════════

  1) 이 폴더 안의 [config.py] 파일 찾기
  2) 마우스 오른쪽 클릭 → 메모장으로 열기
  3) 변경할 API 키를 따옴표 안의 값만 교체
  4) 저장 후 웹앱 재시작

  변경 가능 항목:
  - ANTHROPIC_API_KEY     Claude AI 이미지 분석
  - NAVER_CLIENT_ID/SECRET  네이버 쇼핑 검색
  - COUPANG_ACCESS_KEY/SECRET  쿠팡 OPEN API
  - GEMINI_API_KEY        Google Gemini 이미지 생성
