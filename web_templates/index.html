📂 이미지 폴더 선택

매입 제품 이미지가 들어있는 폴더를 선택해 주세요.

🔍 제품 분류

이미지를 분석해서 제품별로 자동 분류합니다.

📋 분류된 제품

📊 엑셀 자동 생성

제품 정보 + 썸네일 포함 엑셀 파일을 생성합니다. (28개 컬럼: 기본9+상세7+마케팅3+가격3+쿠팡2+마진4)

🔍 네이버 최저가 조회

네이버 쇼핑 API로 각 제품의 온라인 최저가를 조회하고 엑셀에 자동 입력합니다.

🛒 쿠팡 판매 확인 + 마진 분석

쿠팡 판매 여부를 확인하고, 매입가 대비 네이버/쿠팡 마진율을 자동 계산합니다.

확인 방식: 네이버 쇼핑 API에서 쿠팡 판매 여부 + 최저가 자동 추출
마진 계산: 네이버 수수료 6% / 쿠팡 수수료 12% 적용
결과 반영: 엑셀 W~X열 (쿠팡), Y~AB열 (마진 분석)

🎨 상세페이지 AI 이미지 생성

Gemini AI가 제품별 상세페이지 이미지를 자동 생성합니다. 원하는 제품을 선택하세요.

AI 모델 선택
생성 과정: 제품 분석 → 누끼컷 1장 → 상세페이지 6~7장 (후킹/소개/특징/상세/리뷰/갤러리/CTA) → 한장 통이미지
📋 제품 선택
⚠️ Step 1에서 제품을 먼저 분류해 주세요.
💡 후킹 이미지 테스트: 전체 생성 전에 최상단 이미지 1장만 먼저 확인할 수 있어요. 카드에서 제품 1개만 선택 후 🎯 후킹 테스트를 클릭하세요. (약 1~2분 소요)

📦 쿠팡 상품 등록

쿠팡 OPEN API를 통해 상품 등록 데이터(JSON)를 생성합니다.

⚠️ "실제 쿠팡 등록"은 실제로 상품이 등록됩니다. 데이터를 먼저 확인하세요.
